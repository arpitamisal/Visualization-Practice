# Visualization Practice (Python)

This repository contains data analysis and visualization practice using Python. The project includes data cleaning, exploratory data analysis (EDA), feature analysis, and basic machine learning modeling.

## Tech Stack
- Python  
- Pandas & NumPy  
- Matplotlib & Seaborn  
- Scikit-learn  

## Key Work
- Data cleaning and handling missing values  
- Correlation heatmap analysis  
- Token frequency visualization (WordCloud)  
- KNN classification model  
- Model evaluation using accuracy and confusion matrix  

The notebooks demonstrate end-to-end analysis: loading data, preprocessing, visualization, model training, and evaluation.
